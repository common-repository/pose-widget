This plugin gives you a handy widget connected to your Pose account.

== Description ==

The Pose widget allows you to post new style content to your site in real-time with your mobile phone. Pose is a free mobile style app, available at Pose.com, that allows you to create a profile and capture and discover inspiring styles from around the globe. With Pose's widget you can embed your most recent poses directly onto your page. You must have a Pose account to use this widget. Happy Posing!

== Installation ==

1. Upload the `pose` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. From the Widgets page, add the Pose widget to the sidebar of your choice
